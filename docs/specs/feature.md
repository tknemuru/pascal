# 4歳児向けパズルゲーム仕様

## ゲームの概要

* 未就学児を対象とした図形パズルゲームであり、子供の想像力発達を目的としたゲームである。
* iPad/safariでの利用を想定しているpwaである。
* ユーザは与えられた図形をD&Dでターゲット枠にあてはめていく。

## ゲーム全体の流れ

1. 難易度を選択する。
2. 難易度に応じたパズルの型を自動生成する。
3. ユーザがD&Dでパズルの図形をはめていく。
4. すべての図形がはまるとステージクリア。
5. 2-4を5回繰り返す。
6. 5回ステージをクリアしたら、ゲームクリアで終了する。

## 難易度

* 難易度は「かんたん」「ふつう」「むずかしい」の3種類とする。
* かんたん：
  * ターゲット枠とD&D図形の対応が明確にわかる
  * 回転機能なし
  * 初期表示時のイメージは「かんたん-初期表示.HEIC」を参照
* ふつう：
  * ターゲット枠とD&D図形の対応が明確にはわからず、想像力を働かせる必要がある（ターゲット枠の罫線がない状態ということ）
  * 回転機能なし
  * 初期表示時のイメージは「ふつう-初期表示.HEIC」を参照
* むずかしい
  * ターゲット枠とD&D図形の対応が明確にはわからず、想像力を働かせる必要がある（ターゲット枠の罫線がない状態ということ）
  * 図形を必要に応じて回転させないとターゲット枠を埋めることができない
  * 初期表示時のイメージは「むずかしい-初期表示.HEIC」を参照

## D&D図形とターゲット枠（パズルの型）

* D&D図形は以下形の中から選択すること
  * 正方形
  * 長方形
  * 平行四辺形
  * 正三角形
  * 二等辺三角形
* ターゲット枠（パズルの型）はランダムに生成することにより、ユーザを飽きさせないこと

## UX

* 4歳児の女の子が楽しめるように気持ちよくて、派手な演出をする。
  * ターゲット枠にはめたら、子供が喜ぶエフェクトをかけて、Nitendo Switchのような「カチャッ！」という気持ちいい音を出す。
  * ステージ・ゲームクリア時にはクリアした快感が持てる派手な演出をする。
